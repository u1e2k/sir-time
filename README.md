# 🍅 ポモドーロタイマー - My Pomodoro Focus App

集中力を維持し、効率的なタスク管理を支援するポモドーロタイマー Web アプリケーション

**🌐 デモサイト**: https://u1e2k.github.io/sir-time/

![ポモドーロタイマー メイン画面](https://github.com/user-attachments/assets/43eaeb9b-c76f-4a7e-8c13-7379baa08779)

## 概要

高価な物理タイマーの代替となる、カスタマイズ性と記録機能を備えた集中力向上ツールです。

| 項目 | 内容 |
| :--- | :--- |
| **製品名** | My Pomodoro Focus App |
| **目的** | 集中力を維持し、効率的なタスク管理を支援するため、高価な物理タイマーの機能を代替・拡張するWebアプリケーション |
| **ターゲット** | 学生、在宅勤務者、特定のタスクに集中したいすべての人 |
| **動作環境** | 主要なモダンWebブラウザ（Chrome, Firefox, Safariなど）。レスポンシブデザインにより、PC・スマートフォンでの利用に対応 |

## 機能

### タイマー機能 (Timer Module)

- **サイクル管理**: 集中時間と休憩時間を交互に繰り返すポモドーロサイクルを実行
- **時間設定**: 集中時間、短い休憩時間、長い休憩時間、長い休憩を取る間隔（サイクル数）をカスタマイズ可能
  - デフォルト値：集中25分、短休5分、長休15分、4サイクル
- **タイマー制御**: 「開始」「一時停止（中断）」「リセット」ボタン
- **リアルタイム表示**: 残り時間を「分:秒 (MM:SS)」形式で大きく表示し、状態（集中中/休憩中）を明示
- **視覚的表現**: 残り時間を円グラフで視覚的に表現
- **自動開始**: 休憩終了後、自動で次の集中サイクルを開始するオプション

### 通知・アラーム機能 (Notification Module)

- **音声アラーム**: 集中時間または休憩時間が終了した際に、アラーム音を鳴らす
- **デスクトップ通知**: ブラウザが非アクティブな状態でも、タイマー終了を知らせるブラウザ通知（Notification API）
- **通知音選択**: 4種類のアラーム音から選択可能（ベル、チャイム、ジェントル、アラート）

### タスク・記録機能 (Task & Log Module)

- **タスク入力**: 現在の集中サイクルで取り組むタスクを入力する欄
- **ログ記録**: 完了したポモドーロサイクル（タスク名、開始/終了時刻、時間）を自動で記録
- **統計表示**: 日別・週別の「総集中時間」「完了ポモドーロ数」を統計データとして表示（グラフ表示付き）
- **割り込みメモ**: 集中中に思いついた「割り込み事項」を一時的にメモし、集中終了後に確認できるメモ欄

### 設定・カスタマイズ機能 (Configuration Module)

- **基本設定**: 時間設定値を保存し、次回アクセス時も適用
- **パーソナルタスク表示**: 曜日別定期タスクを設定画面で登録し、その日のタスクをアプリ画面に表示
- **テーマ設定**: 5種類のテーマカラーから選択可能（デフォルト、オーシャン、フォレスト、サンセット、ラベンダー）
